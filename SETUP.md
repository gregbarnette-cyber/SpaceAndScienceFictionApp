# Setup Guide

## Requirements

- **Python 3.10+** (developed and tested on Python 3.14.3)

## Python Libraries

| Library | Version | Purpose |
|---|---|---|
| `astroquery` | 0.4.11 | SIMBAD star lookups, NASA Exoplanet Archive queries, JPL Horizons ephemeris, Open Exoplanet Catalogue |
| `astropy` | 7.2.0 | Required by astroquery; astronomical data types and units |
| `requests` | 2.33.1 | Fetches the Exoplanet EU Encyclopaedia CSV directly |
| `pyExoplaneteu` | 1.0.3 | Listed as a dependency but not used directly (CSV format compatibility issue) |

## Installation

1. **Clone or download** the repository to your local machine.

2. **Install dependencies** using pip:

```bash
pip install -r requirements.txt
```

Or install packages individually:

```bash
pip install astroquery astropy requests pyExoplaneteu
```

## Running the Application

```bash
python main.py
```

This launches the interactive main menu in your terminal.

## Data Files

The following CSV files must be present in the project directory for certain features to work:

| File | Used By |
|---|---|
| `propertiesOfMainSequenceStars.csv` | Options 9, 10, 13 — Star System Regions, Main Sequence Star Properties |
| `spTypeHyperLM.csv` | Option 15 — Honorverse Hyper Limits |
| `missionExocat.csv` | Options 2, 5 — Mission Exocat Stars |
| `hwc.csv` | Option 6 — Habitable Worlds Catalog |
| `planetInfo.csv` | Option 12 — Solar System Planets |
| `moonInfo.csv` | Option 12 — Moon Data |
| `dwarfPlanetInfo.csv` | Option 12 — Dwarf Planets |
| `asteroidsInfo.csv` | Option 12 — Major Asteroids |
| `starSystems.csv` | Options 19, 20 — Stars Within a Certain Distance (generated by Option 50) |

> **Note:** `starSystems.csv` is generated by running **Option 50 (Star Systems CSV Query)** from the main menu. All other CSV files are included in the repository.

## Notes

- An internet connection is required for SIMBAD, NASA Exoplanet Archive, JPL Horizons, Open Exoplanet Catalogue, and Exoplanet EU queries.
- The Open Exoplanet Catalogue and Exoplanet EU data are downloaded once per session and cached in memory.
